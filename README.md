This is a repository of all my CAD models.

-----Naming Convention----

Files with the suffix PF are part files created using SolidWorks Files with the suffix SDG are drawing files created using SolidWorks Files with the suffix ADG are SolidWorks drawings that were exported as AutoCAD drawing files. Files with the suffix FCStd are 3D model files created with FreeCAD Files with the suffix PDF are the orthographic drawings of the 3D Models.
